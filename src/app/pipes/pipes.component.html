Pipes in Angular

What are Pipes?

Pipes are a feature in Angular that allows you to transform data directly in your templates. They are used to format data, such as displaying dates, currencies, or any other data transformations. Pipes are simple functions that accept an input value and return a transformed value.

Built-in Pipes

Angular provides several built-in pipes that can be used to transform data in your templates.

Commonly Used Built-in Pipes

  • DatePipe: Formats date values.
  • UpperCasePipe: Transforms text to uppercase.
  • LowerCasePipe: Transforms text to lowercase.
  • CurrencyPipe: Formats numbers as currency.
  • DecimalPipe: Formats numbers with decimal points.
  • PercentPipe: Formats numbers as percentages.

Examples of Built-in Pipes

<p>{{ today | date }}</p>
<p>{{ 'hello world' | uppercase }}</p>
<p>{{ 1234.56 | currency:'USD' }}</p>

In the above examples, the DatePipe formats the current date, the UpperCasePipe transforms text to uppercase, and the CurrencyPipe formats a number as USD currency.
